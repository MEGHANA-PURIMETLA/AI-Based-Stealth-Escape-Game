# AI-Based-Stealth-Escape-Game
## 📌 Description
This project is an AI-based stealth game developed using Python and Pygame. The player must reach the exit while avoiding an intelligent guard.

## 🧠 AI Concepts Used
- A* Pathfinding Algorithm
- State-Based AI (Patrol, Chase, Search)
- Vision Detection
- Intelligent Agent Behavior

## 🎮 Features
- Grid-based environment
- Player movement using arrow keys
- Guard AI that detects and chases player
- Move counter
- Start and restart functionality

## 🛠️ Technologies Used
- Python
- Pygame
- Heapq (for A*)

## ▶️ How to Run
1. Install pygame:
   pip install pygame

2. Run the file:
   python main.py

## 📷 Output
Stealth game where player escapes from guard using AI logic.
